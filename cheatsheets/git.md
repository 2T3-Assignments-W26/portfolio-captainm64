Let's configure the editor to VS code (already installed in labs), run the following in the command line: 

git config --global core.editor "code --wait" 

 

Or if you prefer to use notepad on Windows, run the following command: 

git config --global core.editor "notepad" 

  

On macOS, use the following command: 
git config --global core.editor "open -e -W -n"
